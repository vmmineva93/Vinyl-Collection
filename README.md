# Vinyl-Collection

# Run the Project

## 1. Install the Dependencies
- Open an integrated terminal in the client folder and type:

```sh
npm install

```

## 2. Run the Server
- Open an integrated terminal in the server folder and type:

```sh
node server

```

## 3. Run the Client
- Open an integrated terminal in the client folder and type:

```sh
npm run dev

```
